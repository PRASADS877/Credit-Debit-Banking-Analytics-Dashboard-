Credit & Debit Banking Analytics Dashboard (Tableau | SQL | Excel / Power Bi) :

This project presents an interactive Credit & Debit Banking Transaction Dashboard developed in Tableau, supported by SQL for data extraction and Excel for preprocessing.
The dashboard provides a complete analytical view of banking performance across multiple banks, branches, and transaction methods.

Project Overview
The purpose of this project is to analyze real-world banking transactions involving credit, debit, bank transfers, and card payments.
The Tableau dashboard helps financial analysts and banking professionals identify:

1)High and low activity periods
2)Customer spending behavior
3)Bank-wise contribution to total transactions
4)Preferred transaction methods
5)Seasonal financial trends

Tools And Technologies Used :
1. Tableau : Dashboard Creation , Visual Analytics , Kpis and interactive elements, visual storytelling
2. Power bi : KPI cards , interactive filters , data modeling ( dax , Relationships) , Drill Down analysis.
3. Excel : Its used cleaning the data ,Formatting & transformations,Initial exploratory analysis, using Pivot table making visuals ,kpis and dashboards
4. SQL:  Data extraction ,writing the query using KPIs , filtering , grouping and aggregation

   Conclusion : The analysis shows balanced credit and debit activity across all banks, with steady monthly trends and consistent customer behavior. Transaction methods—credit card, debit card, and bank transfer—are nearly equally used, indicating diverse customer preferences. Overall, the dashboard provides a clear financial overview and supports informed decision-making.
